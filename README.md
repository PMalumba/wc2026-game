# ⚽ FIFA World Cup 2026 - Prediction Game

A multiplayer prediction game for up to 150 players. Predict match winners across every stage of the FIFA World Cup 2026 and compete on the leaderboard.

## How to Play

1. Open the game link
2. Go to **Make Prediction** tab
3. Type your name and click **Join Game**
4. Select a match date → pick the match → pick the winner → submit
5. Watch the **Leaderboard** update as results come in

## Scoring

| Stage | Points |
|-------|--------|
| Group Stage (Jun 11-27) | 5 pts |
| Round of 32 (Jun 28 - Jul 3) | 10 pts |
| Round of 16 (Jul 4-7) | 15 pts |
| Quarter-Finals (Jul 9-11) | 20 pts |
| Semi-Finals (Jul 14-15) | 25 pts |
| Third Place (Jul 18) | 30 pts |
| Final (Jul 19) | 35 pts |

The player with the most total points at the end wins — regardless of whether they predicted the final correctly.

## Features

- All 104 World Cup matches preloaded by date
- Real-time sync across all players (Firebase)
- Live result fetching from football API
- Knockout fixtures auto-update with real team names
- Admin-protected controls (PIN required)
- Bafana Bafana themed 🇿🇦
